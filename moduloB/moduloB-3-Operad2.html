Operadores Relacionais

Os operadores realacionais do JS são os da maioria das linguagens de programação, sendo eles:

  • > Maior que
  • < Menor que
  • >= Maior ou igual a
  • <= Menor ou igual a
  • == Igual a
  • != Diferente de

Vale citar que qualquer expressão que contenha um operador relacional vai gerar um valor booleano, Verdadeiro ou Falso. Ex:

  • 5 > 2 == True
  • 7 < 4 == False
  • 8 >= 8 == True
  • 9 < 7 + 1 == False
  • 5 == 5 == True
  • 4 != 4 == False

Lembrando, sempre que você tiver em uma expressão operadores relacionais e operadores aritméticos, primeiro serão realizados os operadores airtmáticos!
